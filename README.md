# Ruby on Rails / CRUD
## Installation

#### 1st Step
- Clone repository
```bash
$ git clone https://github.com/hxst1/Ruby-on-Rails-Edu-Ruiz.git
```

#### 2nd Step
- Install dependencies
```bash
$ bundle install
```

### 3rd Step
- Migrate database
```bash
$ rails db:migrate RAILS_ENV=development
```

### 4th Step
- Start server
```bash
$ rails server
```
